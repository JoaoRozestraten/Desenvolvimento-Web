# Lab64
Com o servidor ativo é possível navegar por 3 páginas, http://localhost:3000 que é a principal, http://localhost:3000/about
e http://localhost:3000/upload 
Os uploads feitos são guardados na pasta uploads dentro do lab64. O usuario tem a possibilidade de mandar pelo terminal
ou fazer o upload no formulário
O projeto serve para o aluno se adaptar com o uso do node.js e entender mais sobre o fluxo de arquivos detro dos servidores
