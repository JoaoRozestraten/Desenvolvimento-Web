# Lab64
Esse projeto conta com 3 arquivos html, 1 js, pacotes externos (como .json)
Apos dar o comando node .\server.js sera necessario trocar o link manualmente.
Ou seja, Link + '/' (index.html)||| Link + '/about' (para navegar para outra pagina que e about.html) |||e por ultimo Link +'upload' (para baixar um arquivo.txt)
Todavia, caso a parte depois do '/' seja digitada incorretamente o individuo sera redirecionado para uma pagina de erro 404 feita pelo aluno
